HTML（Hypertext Markup Language）

  • 标题标签

​ <h1>~<h6>

  • 段落标签

    段落之间存在缝隙，独占一行

  • 换行标签

​ <br>

  • 水平线标签

​ <hr>

  • 文本格式化标签

​ <b>/<strong> , <u>/<ins> , <i>/<em> , <s>/<del>

  • 媒体标签

    • 图片标签

      <img src="" alt="">

    • 音频标签

      <audio src="" controls></audio>

    • 视频标签

      <video src="" controls></video>

    • 连接标签

      <a href="">超链接</a>

  • 列表标签

    • 无序列表

      <ul></ul>

    • 有序列表

      <ol></ol>

    • li标签

      <li></li>

    ul和ol标签只允许包含li标签

    li可以包含任何标签

    • 自定义列表

      <dl></dl>

      <dt></dt> : 主题

      <dd></dd> : 针对主题的内容

    dl标签只允许dt/dd标签

    dt/dd标签可以包含任意内容

  • 表格标签

​ <table></table> : 表格整体,包裹多个tr

​ <tr></tr> : 表格每行,包裹多个td

​ <td></td> : 表格单元格,包裹内容

​ <caption></caption> : 写在table内部

​ <th></th> : 常用于表格第一行, 替换td

  • 表结构标签

​ <thead></thead>

​ <tbody></tbody>

​ <tfoot></tfoot>

合并单元格

  1. 明确要合并哪几个单元格

  2. 通过左上原则,确定保留谁删除谁

    • 上下合并保留上
    • 左右合并保留左

  3. 给保留的单元格设置属性

  • 表单标签

    • <input>
    • <button>

  • 下拉菜单标签

    <select></select> : 下拉菜单整体

    <option></option> : 下拉菜单每一项

  • 文本域标签

    <textarea></textarea>

  • label

    <label></label> : 绑定内容与表单标签的关系

  • 语义化标签

    • div : 盒子标签,独占一行
    • span : 盒子标签,一行可以显示多个
    • header : 网页头部
    • nav : 网页导航
    • footer : 网页底部
    • aside : 网页侧边栏
    • section : 网页区块
    • article : 网页文章

  • 字符实体

    • &nbsp; : <空格>
    • &lt; : <
    • &gt; : >
    • &amp; : &
    • &quot; : "
    • &apos; : '
    • &cent; : ¢
    • &pound; : £
    • &yen; : ¥
    • &euro; : €
    • &sect; : §
    • &copy; : ©
