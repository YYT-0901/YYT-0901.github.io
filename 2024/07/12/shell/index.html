2024-07-12

shell

  • whoami - 问服务器我是谁，返回用户名

  • pwd - present working directory，返回当前文件夹路径

  • ls - list, 列出当前文件夹下的内容

  • clear - 清空命令界面

  • python3 - python的入口,后面跟上文件表示执行文件

  • ↑ - 键盘键, 可以看回上一条命令

  • ↓ - 对应上面

  • ctrl + a - 光标跳去开头

  • ctrl + e - 光标跳去结尾

  • vi {file} - 进入编辑器，空格后面可以跟上文件名

  • ls -lh {file} - 查看文件信息

  • cat - concatenate(拼接),查看文件内容

  • cp {file1} {file2} - 复制文件

  • rm {file} - 删除文件

  • mkdir [-p] {folder} - 创建目录

    -p 加上表示创建多级目录

    例子: mkdir -p folder1/folder2/folder3

python3

  • quit() - 退出(ctrl + d)
  • help(arg) - 返回arg函数的说明

vim

  • i / a / o - 正常模式下点击进入插入模式

  • Shift + ; (:) - 正常模式进入底层命令模式

  • [n]yy - 复制[n]行内容,默认1

  • [n]p - 粘贴[n]次内容

  • dd - 删除一行内容

  • gg - 光标跳到第一行

  • GG - 光标跳到最后一行

  • / + [查找的单词][\c] - 查找单词,区分大小写(使用n/N来看下一个匹配的单词)

    / - 向下查找

    ? - 向上查找

    \c - 加上表示不区分大小写

  • u - 撤销

插入模式

  • Esc - 插入模式下点击返回正常模式

底层命令

  • Esc - 底层命令模式下点击返回正常模式

  • w - 保存文件

  • q - 返回shell

  • help - 获取帮助

  • set number / set nu - 显示行号

  • set nonumber - 关闭显示行号

  • [n] - 光标跳转到第[n]行

  • [],[]s/[]/[]/[g] - 替换文本

    将第n1行到第n2的old替换成new

    g - 加上表示这一行上的所有

    不写n1,n2则作用在光标行

    可以使用 1,$ 表示全部行
