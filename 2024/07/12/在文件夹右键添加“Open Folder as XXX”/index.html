在文件夹右键添加“Open Folder as XXX”

  1. 打开注册表，在目录（\HKEY_CLASSES_ROOT\Directory\shell）
  2. 右键点击shell -> 新建项 -> 起名XXX
  3. 右方的默认字符串，修改成“Open Folder as XXX”
  4. 右击空白处，新建字符串值命名为Icon,修改字符串值为XXX.exe所在的完整目录

  1. 右键点击刚刚创建好的XXX项再进行新建项，项名称起为command
  2. 修改字符串值为XXX.exe所在的完整目录 + “%V”
