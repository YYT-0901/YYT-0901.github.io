JAVA反射

2024-07-31
未分类
6.7k 词

允许在运行时检查和操作类、方法、字段等结构。反射机制主要包含在 java.lang.reflect 包中，它使得程序可以在运行时动态地获取类的信息，并且调用对象的方法或访问对象的字段。

  1. 获取class对象
  2. 在这个class对象再获取Field，Constuctor，Method对象
  3. 再获取这些对象的所有信息（扒干扒净）

获取class对象(字节码文件对象)的三种方法

  1. Class.forName("全类名")



  2. 类名.class



  3. 对象.getClass()

class类中获取构造方法的方法

  1. Constructor<?>[] getContructors() – 返回所有公共(public)构造方法对象的数组
  2. Constructor<?>[] getDeclaredConstuctors() – 返回所有构造方法对象的数组
  3. Constructor<T> getContructor(class<?> ...parameterTypes) – 获取指定公共构造方法
  4. Constructor<T> getDeclaredContructor(class<?> ...parameterTypes) – 获取指定构造方法

假设存在BigStar类

有构造函数

public BigStar()

public BigStar(String name)

protected BigStar(int age)

private BigStar(String name, int age)

class类中获取字段的方法

  1. Field<?>[] getFields() – 返回所有公共字段
  2. Field<?>[] getDeclaredFields() – 返回所有字段
  3. Field<T> getField(String name) – 获取指定公共字段
  4. Field<T> getDeclaredField(String name) – 获取指定字段

假设存在BigStar类

有字段

private String name

private int age

public String gender

class类中获取成员方法的方法

  1. Method<?>[] getMethods() – 返回所有公共成员方法(包含父类的公共成员方法)
  2. Method<?>[] getDeclaredMethods() – 返回所有成员方法
  3. Method<T> getMethod(String name, class<?> ...parameterTypes) – 获取指定公共成员方法
  4. Method<T> getDeclaredMethod(String name, class<?> ...parameterTypes) – 获取指定成员方法
